# PMU Mentor (Prototype)

## Overview
PMU Mentor is a simple Java console-based prototype developed for the Principles of Software Engineering course.
The prototype demonstrates core mentoring platform functions such as user account creation, login, profile update, and event management.

## Key Features
- Create Account
- Login / Logout
- Update Account Information
- Create Mentoring Event (Virtual / In-person)
- View All Events
- View My Events

## Project Scope
This is a prototype focused on demonstrating software engineering concepts (requirements, design, implementation, testing) and Agile tool usage (Jira, Confluence, GitHub). It is not a full production system.

## Technologies
- Java (Console)
- Jira (Agile tracking)
- Confluence (Project documentation)
- GitHub (Version control)

## How to Run
1. Open the project in any Java IDE
2. Run `Main.java`
3. Use the console menu to create an account, login, and manage events
